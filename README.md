ScalaFX Stats
-------------
>Statistics app using ScalaFx and Scala 3, covering:

* median
* mode
* range
* histogram
* variance
* standardDeviation
* standardError
* covariance
* correlationCoefficient
* centroid
* factorial
* fibonacci
* combinations
* permutations
* probability

Test
----
sbt clean test

Run
---
sbt run
